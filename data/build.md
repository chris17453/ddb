# ddb build 

## Prerequisites (Fedora)

- yum install -y python2-devel       # for cython deps
- yum install -y redhat-rpm-config   # for cython deps
- yum install -y libyaml-devel       # for c bindings on yaml reader 
- uses python 2.7

## Install

```bash
pip install ddb
```

OR

```bash
pipenv install ddb
```

## Dev

- use the makefile
- make build.. etc
